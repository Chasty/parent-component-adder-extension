# addparentcomponent README

This extensions helps you to add a parent component and you can give any name.

## Features

Select a range of code and right-click to see the option `Ã€dd Parent Component` in the context menu. Click on it and you will be prompted to enter the name of the parent component. Enter the name and click on `OK`. The parent component will be created and the selected code will be moved to the parent component.
